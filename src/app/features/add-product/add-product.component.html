Formulario de Registro

El ID es obligatorio. El ID debe tener al menos 3 caracteres. El ID no puede tener más de 10 caracteres. El ID ya está en uso.
El nombre es obligatorio. El nombre debe tener al menos 5 caracteres. El nombre no puede tener más de 100 caracteres.
La descripción es obligatoria. La descripción debe tener al menos 10 caracteres. La descripción no puede tener más de 200 caracteres.
El logo es obligatorio.
La fecha de liberación es obligatoria. La fecha de liberación debe ser una fecha futura.
La fecha de revisión es obligatoria.
